DINOLOOP
REST API Library for building scalable Node.js server-side applications powered by Typescript.

  • Reuse OOP Practises in Node.js
  • SOLID principles and Dependency Injection
  • An Expressjs library influenced by MVC
  • Reuse Expressjs Middlewares and codebase
  • Developers can mix Node.js, Expressjs and Dinoloop
  • Extensive support from Expressjs community
