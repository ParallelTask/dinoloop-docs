DINOLOOP
A lightweight REST API Library for building scalable Node.js server-side applications powered by Typescript.

  • Reuse OOP Practices in Node.js
  • SOLID principles and Dependency Injection
  • An Expressjs library influenced by MVC architecture
  • Isolation of Controllers and Services for testability
  • Write/Reuse Node.js and Expressjs codebase
  • Highly decoupled components for better unit and e2e testing
